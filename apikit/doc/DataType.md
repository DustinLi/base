# 数据类型

## 1. 基本数据类型

    1. boolean
    2. int (32位有符号整数)
    3. long (64位有符号整数)
    4. float (32位浮点数)
    5. double (64位浮点数)
    6. String 
    7. byte[]
    8. Date
    9. enum 枚举类型，只支持简单枚举类型
    10. 数组，以上类型加[]
    
## 2. 类型映射

    协议类型     | java 类型  | swift类型
    :----------:|-----------|----
    boolean     | boolean   | c
    int         | int       | f
    long        | long      | i
    float       | float     | i
    double      | double    | i
    String      | String    | i
    byte[]      | byte[]    | i
    Date        | Date      | i
    enum        | enum      | i
    数组        | List<T>  | Array
